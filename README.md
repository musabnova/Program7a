# Program 7a - JSP Cookie Management

## Objective
Create a JSP program to manage cookies by taking user input and displaying cookie information.

## Files
- index.html
- welcome.jsp
- second.jsp

## Source Code
- HTML: https://github.com/musabnova/Program7a/blob/master/src/main/webapp/index.html
- welcome.jsp: https://github.com/musabnova/Program7a/blob/master/src/main/webapp/welcome.jsp
- second.jsp: https://github.com/musabnova/Program7a/blob/master/src/main/webapp/second.jsp

## How to Run
1. Run project on Apache Tomcat
2. Open: http://localhost:8080/Program7a/
3. Enter details and submit
4. Cookie will be created and displayed
